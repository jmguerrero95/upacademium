Este modulo le permite al Admnistrador del sistema crear, modificar, activar o desactivar perfiles y usuarios para el acceso controlado de los usuarios al sistema, se debe destacar que el sistema registra todas y cada una de las acciones que los usuarios realizan en el sistema, el Administrador por medio de este modulo puede revisarlas y conocer fechas, horas, acciones realizadas en los datos. Igualmente con este modulo se puede registrar datos de base para todo el sistema como paises, ciudades provincias entre otros.
